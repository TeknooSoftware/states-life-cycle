\Teknoo\States\LifeCycle\Scenario

\Teknoo\States\LifeCycle\ScenarioScenarioBuilder

Class ScenarioBuilder Default implementation of scenario builder to allow developper to create scenarii about stated class instances.

A scenario can accept several conditions : each condition must be validated to execute the scenario.
